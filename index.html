Rang d'une matrice

Définition

Le rang de A, noté rg(A), est la dimension de l'espace engendré par les colonnes (ou lignes) de A.

rg(A) = dim(Im(f_A))
Propriétés clés

Pour A ∈ M_{p,q}(ℝ) :

rg(A) ≤ min(p, q) rg(A) = rg(ᵗA) A inversible ⟺ rg(A) = n

Matrice d'une application linéaire

Règle de construction

La j-ème colonne de M(f, B, B') = coordonnées de f(eⱼ) dans B'.

Construction colonne par colonne
  1. Choisir une base B = (e₁,...,eₙ) de E et B' = (f₁,...,fₘ) de F.
  2. Calculer f(eⱼ) pour chaque vecteur de base.
  3. Écrire f(eⱼ) dans la base B' : f(eⱼ) = a₁ⱼf₁ + ··· + aₘⱼfₘ.
  4. La j-ème colonne de M(f, B, B') est le vecteur (a₁ⱼ,...,aₘⱼ).
Si f : E → F, g : F → G alors M(g∘f) = M(g) · M(f)

Théorème du rang

Théorème fondamental
dim Ker(f) + dim Im(f) = dim E

f injective ⟺ Ker(f) = {0}  ·  f surjective ⟺ Im(f) = F  ·  f bijective ⟺ inj + surj

Changement de base

Matrice de passage

P = P_{B→B'} : j-ème colonne = coordonnées de e'ⱼ dans B.

X = P · X' (changement de coordonnées)
Formule de changement

Pour un endomorphisme f : E → E :

M(f, B') = P⁻¹ · M(f, B) · P
Méthode : Construire P → calculer P⁻¹ → appliquer P⁻¹AP. Matrices semblables conservent det et trace.

Matrice de passage P_{B→B'}

Définition — PB→B'

La j-ème colonne = coordonnées de e'ⱼ exprimées dans B.

P_{B→B'} = M(Id_E, B', B)
⚠ Piège fréquent : On exprime les vecteurs de B' dans B (pas l'inverse). Toujours vérifier l'ordre !
Schéma — Rôle de la matrice de passage

Changement de base — Formule centrale

⭐ Théorème à maîtriser — Endomorphisme f : E → E
M(f, B') = P⁻¹ · M(f, B) · P où P = P_{B→B'}
Schéma — Diagramme commutatif

Méthode pas-à-pas

  1. Calculer M(f, B) dans la base canonique : appliquer f à chaque vecteur eⱼ et lire les coordonnées → colonnes.
  2. Vérifier que B' est une base : système libre → det ≠ 0 (carré) ou rang = dimension.
  3. Construire P = P_{B→B'} : exprimer chaque e'ⱼ dans B → j-ème colonne de P.
  4. Calculer P⁻¹ = (1/det P) · Com(P)ᵀ  (ou substitution si P triangulaire).
  5. Appliquer : M(f, B') = P⁻¹ · M(f, B) · P
Exemple complet

f : ℝ₂[X] → ℝ₂[X], B' = (1, X−2, (X−2)²)

M(f,B) = ⎡ 1 −1 1⎤ P_{B→B'} = ⎡ 1 −2 4⎤ P⁻¹ = ⎡1 2 4⎤ ⎢ 1 1 0⎥ ⎢ 0 1 −4⎥ ⎢0 1 4⎥ ⎣ 1 0 −2⎦ ⎣ 0 0 1⎦ ⎣0 0 1⎦ M(f,B') = P⁻¹·M(f,B)·P = ⎡ 7 −13 17⎤ ⎢ 5 −9 8⎥ ⎣ 1 −2 2⎦

Déterminant d'ordre 2

Définition
det A = |a₁₁ a₁₂| |a₂₁ a₂₂| = a₁₁·a₂₂ - a₁₂·a₂₁
Exemples
|1 3| |2 4| = 4 - 6 = -2 |2 4| |1 3| = 6 - 4 = 2

Déterminant d'ordre 3

Développement selon la 1ère colonne
det A = a₁₁·|a₂₂ a₂₃| - a₂₁·|a₁₂ a₁₃| + a₃₁·|a₁₂ a₁₃| |a₃₂ a₃₃| |a₃₂ a₃₃| |a₂₂ a₂₃|
Astuce : Développer selon la ligne/colonne qui a le plus de zéros.

Règle de Cramer & inverse par cofacteurs
🇫🇷 Cours en français

8 Propriétés essentielles

  1. Permuter deux lignes (ou colonnes) multiplie le déterminant par −1.
  2. det(A) = det(ᵗA) : la transposition conserve la valeur.
  3. Multiplier une ligne par k multiplie le déterminant par k.
  4. Si tous les éléments d'une ligne sont nuls → det = 0.
  5. Si deux lignes sont proportionnelles → det = 0.
  6. Ajouter à une ligne une combinaison linéaire des autres → det inchangé.
  7. det(AB) = det(A) · det(B).
  8. det(A⁻¹) = 1 / det(A) si A inversible.

Calcul de l'inverse (méthode des cofacteurs)

Formule
A⁻¹ = (1/det A) · adj(A) où adj(A) = ᵗ[Cᵢⱼ]
Exemple — A = [[2,3,-4],[0,-4,2],[1,-1,5]]
det A = -46 ≠ 0 → A est inversible Cofacteurs : C₁₁=-18, C₁₂=2, C₁₃=4 C₂₁=-11, C₂₂=14, C₂₃=5 C₃₁=-10, C₃₂=-4, C₃₃=-8 A⁻¹ = (-1/46) · [[-18 -11 -10] [ 2 14 -4] [ 4 5 -8]]

Règle de Cramer

Résolution de AX = b (si det A ≠ 0)
xᵢ = det(Bᵢ) / det(A) où Bᵢ = A avec la i-ème colonne remplacée par b
La méthode de Gauss reste la plus efficace pour les grands systèmes. Cramer est surtout utile pour les systèmes 2×2 ou 3×3.

Exercice 1 Matrices commutant avec A = [[a,b],[0,a]] +

Soient a et b ≠ 0 deux réels. A = [[a,b],[0,a]]. Trouver toutes les matrices B ∈ M₂(ℝ) telles que AB = BA.

Question unique

Déterminer l'ensemble des matrices B qui commutent avec A.

On pose B = [[p,q],[r,s]]. On calcule AB et BA et on égalise :

AB = BA impose : br = 0 → r = 0 (car b ≠ 0) b(s-p) = a(s-q) → avec r=0 : p = s (lib.) q libre
B = [[p, q],[0, p]] avec p, q ∈ ℝ quelconques B = p·I + q·N où N = [[0,1],[0,0]] → Les commutants de A sont les polynômes en A.
