01

Rang d'une matrice

Définition

Le rang = nombre de vecteurs colonnes linéairement indépendants.

rg(A) = rg{C₁, …, Cq}
Propriétés essentielles
rg(A) ≤ min(p, q)
A inversible ⟺ rg(A)=n ⟺ det(A)≠0
rg(A) = rg(Aᵀ)
Schéma interactif — Pivot de Gauss étape par étape

Deux pivots identifiés. Les lignes nulles apparaissent.

02

Matrice d'une application linéaire

Règle de construction

La j-ème colonne de M(f, B, B') = coordonnées de f(eⱼ) dans B'.

Schéma — Construction de M(f, B, B')
Somme & scalaire
M(f+g) = M(f) + M(g) M(λf) = λ · M(f)
Composition (ordre !)
M(g∘f, B, B'') = M(g,B',B'') × M(f,B,B')
Application inverse
M(f⁻¹, B', B) = [M(f, B, B')]⁻¹ = (1/det) · Com(M)ᵀ

Schéma — Noyau et Image d'une application linéaire
Noyau

Ker(f) = {X ∈ E : AX = 0}

Résoudre le système homogène → base de Ker(f).

Image

Im(f) = vect des colonnes de A

dim(Im f) = rg(A)
04

Matrice de passage

Définition — PB→B'

La j-ème colonne = coordonnées de e'ⱼ exprimées dans B.

P_{B→B'} = M(Id_E, B', B)
⚠ Piège fréquent : On exprime les vecteurs de B' dans B (pas l'inverse). Toujours vérifier l'ordre !
Schéma — Rôle de la matrice de passage
Changement de coordonnées
X = P_{B→B'} · X'

X = coords dans B  |  X' = coords dans B'

P est toujours inversible
P_{B→B'}⁻¹ = P_{B'→B} P_{B→B'} · P_{B'→B} = Iₙ

05

Changement de bases — Formule centrale

⭐ Théorème à maîtriser — Endomorphisme f : E → E
M(f, B') = P⁻¹ · M(f, B) · P où P = P_{B→B'}
Schéma — Diagramme commutatif
Matrices semblables

M' = P⁻¹MP — même endomorphisme, bases différentes.

det(M') = det(M) tr(M') = tr(M)
Matrices équivalentes

M' = Q⁻¹MP — même application linéaire, 4 bases.

rg(M') = rg(M)
06

Méthode pas-à-pas (exercice type)

  1. Calculer M(f, B) dans la base canonique : appliquer f à chaque vecteur eⱼ et lire les coordonnées → colonnes.
  2. Vérifier que B' est une base : système libre → det ≠ 0 (carré) ou rang = dimension.
  3. Construire P = P_{B→B'} : exprimer chaque e'ⱼ dans B → j-ème colonne de P.
  4. Calculer P⁻¹ = (1/det P) · Com(P)ᵀ  (ou substitution si P triangulaire).
  5. Appliquer : M(f, B') = P⁻¹ · M(f, B) · P
Exemple complet — Application 2 du cours

f : ℝ₂[X] → ℝ₂[X], B' = (1, X−2, (X−2)²)

M(f,B) = ⎡ 1 −1 1⎤ P_{B→B'} = ⎡ 1 −2 4⎤ P⁻¹ = ⎡1 2 4⎤ ⎢ 1 1 0⎥ ⎢ 0 1 −4⎥ ⎢0 1 4⎥ ⎣ 1 0 −2⎦ ⎣ 0 0 1⎦ ⎣0 0 1⎦ M(f,B') = P⁻¹·M(f,B)·P = ⎡ 7 −13 17⎤ ⎢ 5 −9 8⎥ ⎣ 1 −2 2⎦
